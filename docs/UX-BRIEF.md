# 서울이도치과의원 홈페이지 UX 설계 브리프

> 작성 목적: 홈페이지가 단순 정보 전달이 아니라 **환자 경험의 시작점**임을 클라이언트·디자이너와 공유하기 위한 문서.
> 핵심 전제: 병원 방문 전 첫 접점 = 홈페이지. 이도치과의 진료 철학("설명하고 납득시키는 진료")이 사이트에서도 그대로 느껴져야 한다.

---

## 1. 헤더 & 네비게이션

```
┌─────────────────────────────────────────────────────────────┐
│  [YI/DO 모노그램]  소개  의료진  케어  진료  방문   [예약 문의] │
└─────────────────────────────────────────────────────────────┘
                                                         ▲ 주황 버튼
```

| 요소 | 위치 | 이유 |
|---|---|---|
| 로고(모노그램) | 좌상단 고정 | F-패턴 시선 흐름. 브랜드가 항상 시선의 첫 앵커 |
| 내비 링크 | 중앙 | 5개 섹션 직접 접근 — 스크롤 없이 원하는 곳으로 |
| `예약 문의` 버튼 | 우상단 | CTA 하나로 단일화. 주황(`#e95e27`) = 페이지에서 유일한 액센트 컬러 |
| 배경 투명 → YIDO Black | 스크롤 시 자동 | 히어로 영상 위에서도 가독성 유지 |

**환자 경험 관점:** 처음 방문한 환자가 가장 먼저 찾는 행동 = "예약할 수 있나?" — 항상 우측 상단에 고정해 즉시 접근 가능.

## 2. 히어로 섹션 — "이 병원이 어떤 곳인지" 1초 안에

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│   [배경 영상 — 치료 장면 없는 공간·빛·분위기 영상]               │
│                                                              │
│   THE RIGHT WAY TO PERFECTION            ← eyebrow          │
│                                                              │
│   바른 길을 걷다,                           ← H1 대형 타이포   │
│   완벽에 닿다.                               (Gowun Dodum)   │
│                                                              │
│   원칙을 지키는 진단과 바른 치료.               ← lead 문장    │
│   첫 걸음부터 마지막까지. 서울이도치과의원                       │
│                                                              │
│   [ 예약·문의 ▶ ]      [ 이도치과 스토리 → ]                   │
│    ▲ Primary CTA         ▲ Secondary CTA                    │
│                                                              │
│                    ↓ scroll                                  │
└──────────────────────────────────────────────────────────────┘
```

| 요소 | 설계 의도 |
|---|---|
| 배경 영상 — 공간·빛만 | 시술 장면 없음 → 의료광고법 리스크 0, 브랜드 무드 집중 전달 |
| 슬로건 먼저 | 수치·자랑 없이 "어떤 병원인가"를 인상적으로 전달 |
| CTA 2개 분리 | Primary(예약): 바로 결정한 사람 / Secondary(스토리): 더 알고 싶은 사람 |
| Scroll 큐 | 더 볼 내용이 있음을 암시 — 이탈률 감소 |

## 3. 트러스트 스트립 — "믿음의 단서를 빠르게"

```
─────────────────────────────────────────────────────────────
[ 원칙을 지키는 진단 ] · [ 설명하고 동의 ] · [ 철저한 위생·소독 ] · [ 다국어 안내 준비 ]
─────────────────────────────────────────────────────────────
```

**위치:** 히어로 바로 아래, 스크롤 첫 도착점.
**이유:** 치과 첫 방문 환자의 가장 큰 불안 = "믿을 수 있는 곳인가?" — 아이콘 없이 텍스트 칩으로 빠르게 4가지 신뢰 단서 제공.

## 5. 의료진 섹션

```
┌────────────────┐  ┌────────────────┐  ┌────────────────┐
│  [촬영 예정]    │  │  [촬영 예정]    │  │  [촬영 예정]    │
│                │  │                │  │                │
│  의료진 소개    │  │  의료진 소개    │  │  의료진 소개    │
│  준비 중       │  │  준비 중       │  │  준비 중       │
│                │  │                │  │                │
│ "충분히 듣고    │  │ "꼭 필요한 치료  │  │ "1mm의 차이까지 │
│  설명합니다."   │  │  신중하게."     │  │  살핍니다."     │
└────────────────┘  └────────────────┘  └────────────────┘

  의료진 소개는 개원 준비와 함께 순차적으로 공개될 예정입니다.
```

| 결정 | 이유 |
|---|---|
| 3인 카드 (4번째 제거) | 확정 의료진 3인에 맞춘 정직한 구조 — 빈 카드는 "준비 안 됨" 신호 |
| 인용구 배치 | 사진 없어도 철학이 전달됨 |
| placeholder 문구 | 개원 후 사진·학력 확정 시 채우기만 하면 레이아웃 변경 없음 |

## 7. YIDO PASSION STANDARD 7

```
  YIDO PASSION STANDARD 7
  이도치과가 지키는 일곱 가지 기준
  ─────────────────────────────────────────────────────

  1. 가장 빠른 선택보다 바른 한 걸음
  2. 불필요한 과잉진료 없이, 눈높이에서 설명
  3. 진료의 매 순간, 안정감과 편안함
  4. 가족을 돌보는 마음으로 두려움을 살핌
  5. 1:1 맞춤 진료 설계
  6. 까다로운 고난도 재치료, 섬세하게
  7. 대한민국 치과의 바른 기준 지향
```

**위치:** 의료진 섹션 다음.  
**이유:** "이 사람들이 → 이런 철학을 가졌다"는 흐름으로 자연스럽게 연결.  
세로 스크롤로 읽히는 7개 줄글 = 항목 나열이 아닌 리듬 있는 읽기 경험.

## 8. 환자 경험 (THE YIDO EXPERIENCE)

```
  ① 도착과 맞이
       ↓
  ② 편안한 대기
       ↓
  ③ 충분한 상담         ← 탭/클릭 시 설명 펼침
       ↓
  ④ 정성스러운 진료
       ↓
  ⑤ 사후 케어
       ↓
  ⑥ 배웅과 일상 관리

  [선택 단계 설명 예시]
  "들어서는 순간의 공기와 향, 직원의 안내까지 — 첫인상을 정성껏 준비합니다."
```

| 기능 | 이유 |
|---|---|
| 6단계 Journey 타임라인 | 치료 결과가 아닌 **과정과 경험** 시각화 — 이도치과 핵심 차별점 |
| 탭/클릭 시 설명 펼침 | 관심 있는 단계만 더 읽는 능동적 탐색 유도 |
| "내가 이 병원에 가면 어떤 일이 벌어지나" | 첫 방문 불안 선제 해소 |

## 9. 방문 & 예약 섹션

```
┌──────────────────────────────────────────────────────────────┐
│  VISIT US                                                    │
│                                                              │
│  [주소] 개원 시 안내   [전화] 개원 시 안내   [진료시간] 개원 시 안내 │
│                                                              │
│  예약·문의                  오시는 길              진료시간     │
│  전화·네이버 예약·           [지도 임베드 영역]     개원 시      │
│  카카오톡 채널로                                   안내 예정    │
│  예약 준비 중입니다.                                             │
│                                                              │
│  [ 카카오톡 채널 바로가기 ▶ ]                                   │
└──────────────────────────────────────────────────────────────┘
```

| 결정 | 이유 |
|---|---|
| 가짜 진료시간 테이블 제거 | 미확정 정보 제거 — 개원 전 placeholder로 신뢰도 유지 |
| "준비 중입니다" 문구 | 사용 불가 채널처럼 보이지 않도록 능동적 표현 |
| 지도 임베드 영역 | 개원 후 네이버/카카오맵 연동 예정 위치 (현재 placeholder) |

## 10. FAQ 아코디언

```
  ▼ 어떤 진료를 받을 수 있나요?
     임플란트, 심미치료, 보철, 교정·인비절라인, 턱관절·수면치료, 예방관리...

  ▼ 외국어 안내가 가능한가요?
     영어·일본어·중국어 안내를 준비하고 있습니다.

  ▼ YIDO Total Care는 무엇인가요?
     ...
```

**이유:** 처음 방문 전 환자가 가장 많이 걱정하는 질문을 미리 선제 해소.  
아코디언 = "부담 없이 궁금한 것만 열어본다" — 텍스트 벽으로 느껴지지 않음.

## 11. 다국어 전환 (EN / JA / ZH)

```
  헤더 우측 언어 선택:  KO  EN  JA  ZH

  [각 언어 클릭 시 해당 언어 페이지로 이동]
```

**이유:** 해외 환자 — 특히 일본·중국어권 환자 타깃 명시. 다국어 페이지는 정적 사전생성으로 SEO/AEO 모두 커버.
